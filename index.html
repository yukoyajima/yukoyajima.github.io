AGM

This assignment included an Instagram story, post, and Facebook banner to promote the AGM. The only guidance I got was "egg!" (because it was held in the nest) so I initially took that in a more general sense and wanted to create 3 completely different pieces showcasing different kinds of eggs. The idea of an egg in itself is about the start of something new, in this case a fresh beginning for this organization.

Listen to CiTR!

Finals for all 4 posters. I tried to match the aesthetics of each poster to the genre it promotes. Until this assignment all of the media I created was more illustrative so this project challenged me. To create readable, text-forward grahpics while keeping everything engaging was initially difficult to grasp, especially considering the differing lengths of the band names and figuring out which types to use (I usually like creating my own but that would have been unreasonable for this project.) Sometimes simple is better.
